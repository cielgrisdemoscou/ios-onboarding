# iOS Onboarding Animation 

### [SwiftUI Lego Style Loading Animation](https://github.com/cielgrisdemoscou/ios-project/tree/master/SwiftUI-LegoStyleLoadingAnimation)

<img src="https://user-images.githubusercontent.com/50784573/104872391-1dcc7280-5991-11eb-8edf-b20e72ed98aa.gif"/>

This is a onboarding & loading animation using 100% SwiftUI inspired by [this post](https://codemyui.com/lego-style-loading-animation/) This box inspired loading animation designed by Chris Gannon shows a bunch of lego blocks moving around.

**Features**

- SwiftUI
- Without 3rd party library
- Clean, scalable and maintainable code
